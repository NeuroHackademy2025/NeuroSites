How to Use NeuroSites

Getting Started

Welcome to NeuroSites! Follow these simple steps to find and use the perfect template for your project:

1. Browse Templates

Visit our Templates page to explore our collection. Use the category tabs to filter templates by type: Lab Websites, Personal Pages, Event Promotions, or Package Wikis. You can also search by keywords or use the dropdown filters for styles and themes.

2. Preview Templates

Click "View Template" to see a live preview of any template. This helps you understand the layout, design, and functionality before downloading.

3. Download and Customize

Click "View Code" to access the template files on GitHub. You can download the entire template folder and customize it with your content, colors, and branding.

4. Host Your Site

Deploy your customized template using GitHub Pages, Netlify, Vercel, or any web hosting service. Most templates include deployment instructions in their README files.

Customization Tips

Basic Customization

  • Content: Replace placeholder text with your own content
  • Images: Add your own photos, logos, and graphics
  • Colors: Modify CSS color variables to match your branding
  • Fonts: Update font families in the CSS to match your preferences

Advanced Customization

  • Layout: Modify HTML structure and CSS grid/flexbox properties
  • Functionality: Add JavaScript for interactive features
  • Responsive Design: Adjust media queries for different screen sizes
  • SEO: Update meta tags, titles, and structured data

Deployment Options

GitHub Pages (Free)

  1. Create a GitHub repository for your site
  2. Upload your customized template files
  3. Enable GitHub Pages in repository settings
  4. Your site will be available at username.github.io/repository-name

Netlify (Free tier available)

  1. Create a Netlify account
  2. Drag and drop your template folder to Netlify
  3. Get an instant live URL
  4. Connect to Git for automatic deployments

Vercel (Free tier available)

  1. Create a Vercel account
  2. Import your project from GitHub
  3. Automatic deployments on every commit
  4. Custom domain support

Need Help?

If you need assistance with any template or have questions about customization, please visit our Support page or check out our GitHub repository for more resources.
