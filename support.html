FAQ

Can I use these templates commercially?

Yes. Templates are under CC BY 4.0—personal and commercial use allowed. Please provide attribution.

How do I customize a template?

Start with our Guide. Edit text in index.html, swap images in assets/images/, and tweak CSS variables for colors.

My images don't show on GitHub Pages

Check file paths and filename case (Pages is case-sensitive). Ensure the images are committed and the path matches the folder structure.

How long do responses take?

We try to acknowledge new issues within 2–3 days and give initial feedback within a week. Critical bugs get priority.
