Support & Help

Found a bug, have a suggestion, or need help with a template? We're here to assist!

Submit an Issue

Use our GitHub Issues page to report problems or request new features:

Submit New Issue

Common Issue Types

  • Bug Reports: Template not working as expected
  • Feature Requests: Ideas for new templates or functionality
  • Template Issues: Problems with specific templates
  • Documentation: Unclear instructions or missing information
  • General Help: Questions about using or customizing templates

Before Submitting an Issue

To help us assist you better, please:

  1. Check if your issue has already been reported in our existing issues
  2. Include as much detail as possible about the problem
  3. Provide steps to reproduce the issue if applicable
  4. Include screenshots or error messages when relevant
  5. Mention which template you're working with

Frequently Asked Questions

How do I customize a template?

Check out our comprehensive Guide page which covers everything from basic customization to deployment options.

Can I use these templates for commercial projects?

Yes! All templates are released under the CC BY 4.0 license, which allows both personal and commercial use. Just make sure to give appropriate credit.

How do I contribute my own template?

Visit our Contribute page for detailed instructions on how to submit your own template to the collection.

I found a bug in a template. How do I report it?

Please submit an issue on GitHub with details about the bug, which template it affects, and steps to reproduce it.

Can I request a new template category or type?

Absolutely! We welcome suggestions for new template categories. Please submit a feature request through our GitHub Issues page.

Response Times

This is a community-driven project, so response times may vary. We do our best to:

  • Acknowledge new issues within 2-3 days
  • Provide initial feedback within a week
  • Address critical bugs as quickly as possible
  • Review and merge contributions promptly
